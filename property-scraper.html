Property Scraper

Source Code: https://github.com/tomcmead/property-scraper

Project Overview
This project scrapes Rightmove for property with the specific criteria the user has selected. Once a new property that matches the criteria a notification is sent to the user. The back-end is a REST API built using Python, Flask and SQLite, which then integrates with a front-end cross platform Flutter application.

Python REST API
The Python REST API is able to add/remove/update users and their rightmove location code. Based on the user criteria, links are returned for available properties within the selected area based on the rightmove location code. Data is stored in a SQLite database.

Flutter Application
The front-end comprises of a cross platform Flutter application written in Dart. The Flutter application integrates the REST API to provide a stylish user-interface.
