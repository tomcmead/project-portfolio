Google Trends Crypto Price Analysis

Source Code: https://github.com/tomcmead/google-trends-crypto-price-analysis

Project Overview
Cryptocurrency price predictions based on Google Trends analysis of the crypto asset. Each tradeable cryptocurrency listed on Binance is analysed. This libary be used for cryptocurrency analysis by specifying parameters such as time period, asset, interest, price and graph generation. Testing has revealed Google Trends interest of a cryptocurrency significently increases before the price increases. From this cryptocurrencies of greater interest are predicted to increase in price, if there is a postive price trend.

Implementation
The project is able to pull all listed crypto asset names and tickers from Binance. The interest of listed assets are analyzed using Google Trends. Analysis of all tradeable cryptocurrencies listed on Binance based on price and interest, generates overlapping graph of price and interest. The project is built with Binance API, pytrends CoinGecko API, pandas and matplotlib.
